# Data Directory

This directory contains all datasets used in the World Cup Fan Intelligence Hub project.

## Structure

- **raw/**: Original, unmodified datasets
  - `international_results.csv`: International football match results (1872-2026)
  
- **processed/**: Cleaned and preprocessed datasets
  - `train_data.csv`: Training dataset with engineered features
  - `test_data.csv`: Test dataset for model evaluation

## Dataset Information

### International Football Results Dataset

**Source:** Kaggle / Public Football Data Repository  
**Records:** 49,329 matches  
**Time Range:** 1872-2026  
**Size:** ~5 MB

**Columns:**
- `date`: Match date (YYYY-MM-DD)
- `home_team`: Home team name
- `away_team`: Away team name
- `home_score`: Goals scored by home team
- `away_score`: Goals scored by away team
- `tournament`: Competition name (e.g., FIFA World Cup)
- `city`: Match location city
- `country`: Host country
- `neutral`: Boolean indicating neutral venue

## Usage

1. Download the dataset from the source
2. Place `international_results.csv` in the `raw/` directory
3. Run the preprocessing script to generate processed datasets

## Notes

- Raw data files are excluded from Git (see .gitignore)
- Processed files are generated by the data preprocessing pipeline
- Always keep raw data unchanged for reproducibility
